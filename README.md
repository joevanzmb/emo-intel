EMO-INTEL ‚Äî AI-Powered Mood & Readiness Scanner

A modern mobile-first prototype for emotional analysis and mental readiness assessment, combining AI-driven insights, CBT micro-interventions, and local wellness recommendations.

<p align="center"> <img src="assets/logo-emo-intel.png" alt="EMO-INTEL Logo" width="120"> </p>

EMO-INTEL is an interactive prototype that simulates an AI-based emotional intelligence system.
This project was designed to demonstrate:

- Mood detection workflow
- Emotion-based readiness scoring
- Personalized interventions
- Herbal & local wisdom recommendations
- Emotional insights dashboard
- Fully mobile-responsive UX

The prototype includes face scan UI, mood sliders, questionnaire, readiness result, interventions, emotional insights, and profile/settings pages.
